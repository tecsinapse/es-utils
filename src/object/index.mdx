# Object Utils API

> ### Note
>
> By default, this functions were tested only in `ES6` environments.<br/>
> All are exported at `/tecsinapse-es-utils/core/object/index.js`


## `cleanArray(a):Array`

`objUtils.cleanArray`

This functions removes any duplicate item at an array.


### Parameters

- a: `[]`<br/>
  The input array.


### Output

- `a` array without duplicates


### Examples

```js
import { cleanArray } from '@tecsinapse/es-utils/core/object'

const arrayTest = [1, 2, 3, 4, 5, 5, 5];

console.log("cleanArray: " +  cleanArray(arrayTest));
```

Expected output: <br/>
- `cleanArray: 1,2,3,4,5`

## `flatMap(f, xs):Array`

`objUtils.flatMap`

This functions flats a array using the 'f' function to determine final itens of the array.


### Parameters

- f: `function`<br/>

```js
const flatArrayTest = [[1, 2], [3, 4], [5, 6]];

console.log("flatMap: " +  flatMap(v => v, flatArrayTest));
```

## `concat(x, y):Array`

`objUtils.concat`

This functions perforns concatenation between two arrays `x` and `y` and return `xy` array.


### Parameters

- x: `[]`<br/>
  First array input

- y: `[]`<br/>

### Output

- `xy` concatened array.

```js
const arrayTestConcat = [6, 7, 8, 9, 10, 11, 12];

console.log("concat: " +  concat(arrayTest, arrayTestConcat));

```

Expected output: <br/>

## `removeDuplicates(myArr, prop):Array`

`objUtils.removeDuplicates`

This function removes the duplicate itens of the field `prop` inside `myArr` array.

### Parameters

- myArr: `[]`<br/>
  The input array.

- prop: `string`<br/>
  The field name wich functions will verify oneness, it's not allow to use deep fields.

### Output

- `myArr` with duplicates removed.


### Examples

```js
import { removeDuplicates } from '@tecsinapse/es-utils/core/object'

const removeDuplicatesTest = [
  {id: 1},
  {id: 1},
  {id: 2}
];

console.log("removeDuplicates: " +  removeDuplicates(removeDuplicatesTest, 'id'));

```

Expected output: <br/>
- `removeDuplicates: [object Object],[object Object]` (it have showed only two of three)

## `omitDeep(obj, key):Array`

`objUtils.omitDeep`

This function removes the field with name `key` inside the `obj` object.

### Parameters

- obj: `{}`<br/>
  The input object.

- key: `string`<br/>
  The field key that will be removed.

### Output

- `obj` with `key` removed.


### Examples

```js
import { omitDeep } from '@tecsinapse/es-utils/core/object'

const omitDeepTest = {
  omit: true,
  notOmit: true
};

let omitDeep1 = omitDeep(omitDeepTest, 'omit');
console.log("omitDeep: " + omitDeep1.omit + " - "  + omitDeep1.notOmit );

```

Expected output: <br/>
- `omitDeep: undefined - true`

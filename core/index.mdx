# Tecsinapse-es-utils

Utilities for Javascript developers:

- **object utils**. Utilities for objects like verif string whether empty or not.
- **log utils**. An error logger for sentry error tracking.
- **dom utils**. Utilities for DOM management.


Any issue concact us at [github](https://github.com/tecsinapse/tecsinapse-es-utils) or [npm](https://www.npmjs.com/package/@tecsinapse/es-utils)


## Installation

Using npm:<br/>
`npm i @tecsinapse/es-utils`
